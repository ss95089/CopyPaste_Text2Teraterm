# CopyPaste_Text2Teraterm

実行コマンドを記載したテキストファイルを読み取り→TeraTermにコピペするツールです。  
コマンドプロンプトからEXEファイルを実行します。


##実行方法
1. コマンドプロンプトを起動  

2. ツールのEXEファイルを実行  
   コマンドのテキストファイルは-fオプションにて指定  
> CopyPaste_Text2Teraterm_vX.X.exe -f [command text file name]
